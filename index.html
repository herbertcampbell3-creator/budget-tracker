💰 Budget Tracker

Connect a free Firebase database so Herb & Leslie
can share expenses in real time.

One-time setup (takes ~3 min)

  1. Go to console.firebase.google.com
  2. Click Create a project → name it anything
  3. Left sidebar → Build → Realtime Database
  4. Click Create database → choose Start in test mode
  5. Left sidebar → Project settings ⚙️ → General
  6. Scroll to Your apps → click </> Web
  7. Register app, then copy the firebaseConfig object
  8. Paste it below:
